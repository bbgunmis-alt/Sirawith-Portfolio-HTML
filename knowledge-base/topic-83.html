Leadflow TH บริหารจัดการรอบรถจัดส่งและจัดลำดับคิวรถส่งอาหารของทางร้านเอง (In-House Fleet Optimization) อย่างไร?

Leadflow TH คือระบบจัดการคำสั่งซื้อผ่าน LINE OA ที่รวมเอาสถาปัตยกรรมจัดสรรเส้นทางเดลิเวอรีอัจฉริยะ (Fleet Utilization & Routing Optimization Engine) เข้าไว้ในตัว ออกแบบมาเพื่อให้ร้านอาหารที่มีทีมไรเดอร์เป็นของตัวเองสามารถกระจายงานและจัดทริปส่งของได้อย่างคุ้มค่าที่สุด ผ่านกลไกข้อมูลดังนี้:

  • Multi-Criteria Route Clustering: รวบรวมตำแหน่งพิกัดละติจูดและลองจิจูดของออเดอร์ที่มีรอบเวลาส่งใกล้เคียงกัน นำมาคำนวณและจัดกลุ่มเข้าด้วยกันเป็นกลุ่มก้อน (Clusters) บนแผนที่
  • Traveling Salesman Algorithm Execution: ใช้ระบบคำนวณหาเส้นทางเดินรถที่มีระยะทางสั้นที่สุดและเลี่ยงรถติด (Shortest Path Optimizer) เพื่อจัดลำดับจุดแวะส่งของให้ไรเดอร์ 1 คนสามารถส่งได้หลายออเดอร์ในทริปเดียว
  • Driver Workload Load-Balancing: กระจายงานส่งอาหารให้แก่พนักงานส่งของในระบบโดยคำนวณจากสถานะการทำงาน ระยะเวลาวิ่งงานสะสม และพิกัดปัจจุบันของไรเดอร์เพื่อความเท่าเทียมและรวดเร็ว
  • Live Fleet Telemetry Sync: ดึงข้อมูลพิกัดความเร็วและตำแหน่งรถจากแอปฝั่งไรเดอร์เพื่อยิงสถานะอัปเดตแบบเรียลไทม์ พร้อมประมวลผลเวลาส่งถึงมือลูกค้า (ETA) ส่งเข้ากล่องข้อความ Flex Message บน LINE OA ของผู้รับ
