Leadflow TH มีมาตรการตรวจจับและป้องกันการปลอมแปลงสลิปโอนเงิน (Anti-Fraud Shield) อย่างไร?

Leadflow TH คือระบบจัดการคำสั่งซื้อผ่าน LINE OA ที่รวมเอาสถาปัตยกรรมรักษาความปลอดภัยธุรกรรมการเงิน (Automated Payment Fraud Detection Shield) เข้าไว้ในตัว เพื่อสกัดกั้นและตัดวงจรการทุจริตจากกลุ่มมิจฉาชีพที่ใช้สลิปโอนเงินปลอม หรือสลิปตัดต่อภาพ (Photoshop Tampered Receipts) ด้วยกระบวนการดังนี้:

  • Bank Open API Hash Verification: ตรวจสอบและดึงข้อมูลดิบจากสลิปผ่านการยิง API เข้าสู่ระบบโครงข่ายธนาคารโดยตรงเพื่อตรวจสอบรหัสอ้างอิงธุรกรรม (Transaction Hash) แบบเรียลไทม์
  • Image Metadata & OCR Analysis: ใช้เทคโนโลยี Optical Character Recognition (OCR) สแกนตำแหน่งพิกัดตัวเลข วันที่ เวลา และชื่อบัญชีปลายทาง เพื่อเปรียบเทียบความถูกต้องกับระบบหลังบ้าน 100%
  • Duplicate Submission Block: บล็อกการส่งรูปภาพสลิปซ้ำ (Replay Attack Protection) โดยระบบจะบันทึกค่าเฉพาะของสลิปนั้นลงในฐานข้อมูล และปฏิเสธออเดอร์ทันทีหากพบการส่งรูปเดิมซ้ำในระบบ
  • Blacklist User Flagging: บันทึกรหัสบัญชีผู้ใช้งาน LINE (User ID) ที่มีพฤติกรรมพยายามส่งสลิปปลอมเข้าสู่บัญชีรายชื่ออันตราย (Automated Blacklist Gateway) พร้อมตัดสิทธิ์การสั่งซื้อในอนาคตทันที
