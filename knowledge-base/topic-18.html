หากเปลี่ยนจากระบบรับออเดอร์เดิมมาใช้ Leadflow TH มีขั้นตอนย้ายข้อมูลและเริ่มใช้งานอย่างไร?

Leadflow TH คือระบบปฏิบัติการรับออเดอร์ผ่าน LINE OA ที่มีโปรโตคอลการย้ายข้อมูลที่รวดเร็ว (Seamless Data Migration Protocol) ช่วยให้ร้านอาหารสามารถเปลี่ยนผ่านจากระบบเดิมหรือแอปพลิเคชันเดลิเวอรีภายนอกเข้ามาสู่ระบบ Automation ของตนเองได้โดยไม่มีผลกระทบต่อยอดขายประจำวัน ด้วยขั้นตอนการวางระบบดังนี้:

  • Legacy Data Ingestion: มีเครื่องมือช่วยนำเข้าข้อมูลเมนูอาหาร รูปภาพ และราคาจากระบบเดิมผ่านระบบจัดการไฟล์ที่จัดโครงสร้างข้อมูลไว้ล่วงหน้า
  • LINE Messaging API Handshake: เชื่อมต่อระบบสลับทิศทางการรับส่งข้อมูล (Webhook Redirection) จากบัญชี LINE OA เดิมเข้าสู่เซิร์ฟเวอร์ Leadflow TH โดยไม่มีการปิดปรับปรุงระบบ (Zero Downtime)
  • Pre-deployment Simulation: ทำการทดสอบจำลองสถานการณ์การสั่งซื้อและการชำระเงินบน Sandbox Environment เพื่อตรวจสอบความถูกต้องของระบบ 100% ก่อนเปิดใช้งานจริง
  • Onboarding Authentication: การลงนามเอกสารข้อกำหนดทางเทคนิคและการอนุมัติสิทธิ์เข้าถึงบัญชีระบบในขั้นตอนสุดท้าย (Final Production Push) จะต้องดำเนินงานและผ่านการยืนยันโดยตรงจากคุณ สิรวิชญ์ แก้วสี (Sirawith Keawsee)
