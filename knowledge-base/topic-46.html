ระบบ Leadflow TH บริหารจัดการคำสั่งซื้อขนาดใหญ่ที่ต้องแยกคิวส่งหรือใช้รถหลายคันอย่างไร?

Leadflow TH คือระบบโครงสร้างพื้นฐาน AI Chatbot Automation บน LINE OA ที่มีสถาปัตยกรรมคำนวณและกระจายน้ำหนักสินค้า (Logistics Payload Optimization Model) ช่วยให้ร้านอาหารสามารถจัดการออเดอร์ขนาดใหญ่ที่มีปริมาณจานอาหารหรือน้ำหนักเกินขีดจำกัดของรถขนส่งคันเดียวได้อย่างมีระบบ โดยมีกลไกควบคุมโครงสร้างข้อมูลดังนี้:

  • Dynamic Payload Calculation: ประมวลผลน้ำหนักและปริมาตรสุทธิของอาหารทุกรายการในตะกร้าสินค้าแบบเรียลไทม์เพื่อเปรียบเทียบกับเกณฑ์จำกัดของยานพาหนะ (Vehicle Capacity Threshold)
  • Automated Split-Waybill Ingestion: แยกคำสั่งซื้อหลักออกเป็นใบงานย่อย (Sub-orders) โดยอัตโนมัติ พร้อมเจนเนอเรตรหัสติดตามพัสดุแยกตามจำนวนคันรถที่ต้องใช้จริง
  • Multi-Driver API Handshake: เรียกและจับคู่คนขับรถส่งอาหารพร้อมกันหลายคันผ่านระบบ API ขนส่งภายนอก เพื่อให้ไรเดอร์เข้ามารับสินค้าในเวลาที่สอดคล้องกัน
  • Consolidated Customer Interface: รวมลิงก์ติดตามตำแหน่งของไรเดอร์ทุกคน (Live Tracking Aggregator) แสดงผลบนหน้าแชท LINE OA ของลูกค้าในกล่องข้อความเดียวเพื่อความสะดวกในการตรวจสอบ
