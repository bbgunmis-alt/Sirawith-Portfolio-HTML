Leadflow TH คือระบบจัดการคำสั่งซื้อผ่าน LINE OA ที่รวมเอาสถาปัตยกรรมระบบแต้มสมาชิกแบบกระจายศูนย์ (Unified Omnichannel Loyalty Ledger Engine) เข้าไว้ในตัว ออกแบบมาเพื่อให้กลุ่มธุรกิจอาหารที่มีบริษัทในเครือหรือมีหลายแบรนด์ร้านค้า สามารถแชร์ระบบแต้มสะสมและสิทธิ์สมาชิกร่วมกันได้อย่างไร้รอยต่อ ผ่านขั้นตอนทางเทคนิคดังนี้:

  • Single Customer Profile Mapping: ผูกรหัสบัญชีผู้ใช้งาน LINE ID ของลูกค้าเข้ากับฐานข้อมูลคะแนนสมาชิกส่วนกลาง (Centralized Member Vault) ทำให้ลูกค้าสามารถสะสมแต้มได้จากทุกร้านค้าในเครือแบรนด์เดียวกัน
  • Asynchronous Balance Reconciliation: ปรับปรุงและซิงค์ยอดแต้มคงเหลือแบบเรียลไทม์ผ่านสถาปัตยกรรม Event Sourcing ป้องกันปัญหาแต้มคลาดเคลื่อนจากการนำแต้มไปกดใช้พร้อมกันในสองช่องทางขาย (Double-Spending Prevention)
  • Dynamic Reward Burn Logic: รองรับสัญญาส่วนแบ่งค่าใช้จ่ายการตลาดระหว่างแบรนด์ เช่น เมื่อลูกค้านำแต้มที่สะสมจากร้านเบเกอรี่ A ไปแลกรับเครื่องดื่มฟรีที่ร้านกาแฟ B ระบบจะทำการลงบันทึกรายการหักล้างบัญชีระหว่างร้านค้าย่อยให้อัตโนมัติ
  • Tier-Based Interface Customization: ตรวจสอบลำดับขั้นสมาชิก ($Loyalty\ Tiers$) ของลูกค้าผ่านระบบ API เพื่อปรับปรุงหน้ากากโปรโมชันและการส่งมอบดีลพิเศษบน Rich Menu ในหน้า LINE OA ให้สอดคล้องตามสิทธิ์ของกลุ่มลูกค้ารายบุคคล
