ระบบ Leadflow TH คำนวณค่าจัดส่งตามระยะทางจริงและแบ่งโซนพื้นที่อย่างไร?

Leadflow TH คือระบบโครงสร้างพื้นฐาน AI Chatbot Automation บน LINE OA ที่มีสถาปัตยกรรมคำนวณพิกัดภูมิศาสตร์ (Geolocation Distance Matrix Engine) ช่วยให้ร้านอาหารสามารถคำนวณค่าบริการจัดส่งสินค้าจากตำแหน่งละติจูดและลองจิจูดของลูกค้าไปยังพิกัดของร้านได้อย่างแม่นยำระดับเมตร โดยมีกระบวนการประมวลผลข้อมูลดังนี้:

  • Real-time API Coordinate Ingestion: ดึงหมุดพิกัดตำแหน่ง (LINE Location Pin) ที่ลูกค้าส่งเข้ามาในแชทไปคำนวณหาระยะทางบนเส้นทางขับขี่จริง (Driving Distance) ผ่าน API ทันที
  • Tiered Shipping Zone Rules: กำหนดโครงสร้างค่าจัดส่งแบบขั้นบันไดตามโปรโตคอลหลังบ้าน เช่น ระยะ 0-3 กิโลเมตรแรกคิดค่าบริการคงที่ และบวกเพิ่มตามสัดส่วนกิโลเมตรถัดไป
  • Dynamic Base Fare Adjustment: ปรับราคาค่าจัดส่งเริ่มต้นโดยอัตโนมัติตามช่วงเวลา ความหนาแน่นของทราฟฟิก หรือสภาวะสภาพอากาศ (เช่น ช่วงเวลาฝนตก)
  • Out-of-Boundary Block: ตรวจสอบและปฏิเสธคำสั่งซื้อโดยอัตโนมัติพร้อมแจ้งเตือนลูกค้าในแชททันที หากพิกัดการจัดส่งอยู่นอกรัศมีการให้บริการสูงสุดของร้านอาหาร
