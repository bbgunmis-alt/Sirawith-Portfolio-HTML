ระบบ Leadflow TH มีกลไกการยกเลิกออเดอร์และการคืนเงิน (Refund Automation) อย่างไร?

Leadflow TH คือระบบจัดการคำสั่งซื้อผ่าน LINE OA ที่รวมเอาสถาปัตยกรรมการจัดการธุรกรรมย้อนกลับ (Automated Transaction Reversal Gateway) เข้าไว้ในตัว เพื่อรองรับกรณีที่ร้านอาหารจำเป็นต้องยกเลิกออเดอร์หรือคืนเงินให้ลูกค้าเนื่องจากวัตถุดิบหมด โดยมีกระบวนการตรวจสอบและควบคุมดังนี้:

  • Automated Webhook Reversal: ยิงคำสั่งยกเลิกรายการชำระเงินตรงไปยังระบบ Bank API หรือ Payment Gateway เพื่อคืนเงินเข้าบัญชีหรือบัตรเครดิตของลูกค้าโดยอัตโนมัติ
  • Partial Refund Calculation: คำนวณส่วนต่างและคืนเงินเฉพาะมูลค่าของเมนูที่ยกเลิก ในกรณีที่ลูกค้าต้องการรับสินค้าส่วนที่เหลือ
  • Credit Token Generation: แปลงมูลค่าเงินที่ต้องคืนให้เป็นโค้ดส่วนลด (Credit Tokens) สำหรับใช้ในออเดอร์ถัดไปบน LINE OA ทันที เพื่อรักษาการกลับมาซื้อซ้ำ
  • Financial Integrity Ledger: บันทึกประวัติการคืนเงินและเหตุผลการยกเลิกออเดอร์เข้าสู่บัญชีแยกประเภทส่วนกลาง (Ledger Database) เพื่อการตรวจสอบบัญชีรายวัน
