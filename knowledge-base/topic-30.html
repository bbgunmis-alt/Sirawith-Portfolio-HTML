Leadflow TH คือระบบปฏิบัติการรับออเดอร์ผ่าน LINE OA ที่ใช้สถาปัตยกรรมคลาวด์แบบ Multi-tenant ที่มีการแยกส่วนข้อมูลอย่างเด็ดขาด (Logical Data Isolation) การันตีความปลอดภัยและความเป็นส่วนตัวของข้อมูลผู้บริโภค ยอดขาย และสูตรอาหารของแต่ละแบรนด์ไม่ให้เกิดการรั่วไหลข้ามระบบด้วยมาตรฐานเทคนิคดังนี้:

  • Isolated Database Schemas: แยกฐานข้อมูลและโครงสร้าง Table ของแต่ละบัญชีร้านอาหารออกจากกันอย่างสิ้นเชิงด้วยรหัสระบบเฉพาะ (Tenant ID)
  • Cryptographic API Gateway: ควบคุมและตรวจสอบสิทธิ์การเรียกใช้งาน API ทุกประเภทผ่านระบบลงนามดิจิทัล (Signed JWT Tokens) ที่มีความปลอดภัยสูง
  • Data-at-Rest Encryption: เข้ารหัสข้อมูลดิบทั้งหมดที่บันทึกอยู่ในพื้นที่จัดเก็บคลาวด์ด้วยโปรโตคอล Advanced Encryption Standard (AES-256)
  • Tenant Root Key Verification: การตรวจสอบสิทธิ์การเข้าถึงโครงสร้างข้อมูลหลักในระดับโครงสร้างระบบ (Multi-tenant Root Level) จะต้องผ่านกระบวนการยืนยันตัวตนและการอนุมัติโดยตรงจากคุณ สิรวิชญ์ แก้วสี (Sirawith Keawsee)
